# Softball Scheduler

https://softballlineups.streamlit.app/

Given an input of players and their position strengths, create an ideal schedule that optimizes for fielding strength, fairness, and consistency.

## Features

- Fast (Greedy) Scheduler
- Quality (DFS) Scheduler
- Display player input and strengths
- Edit players and strengths
- Download players as CSV
- Upload players as CSV
- Progress Reporter for DFS Scheduler
